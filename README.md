#  JavaScript To-Do List Application

A modern, responsive To-Do List application built with **HTML5, CSS3, and JavaScript**. This project demonstrates client-side state management, DOM manipulation, event handling, and data persistence using the browser's Local Storage API.

---

##  Project Overview

The To-Do List application helps users efficiently manage daily tasks by allowing them to create, edit, delete, and mark tasks as completed. All task data is automatically saved in the browser using **Local Storage**, ensuring tasks remain available even after refreshing or reopening the browser.

---

##  Features

-  Add New Tasks
-  Edit Existing Tasks
-  Delete Tasks
-  Mark Tasks as Completed
-  Toggle Task Status
-  Automatic Data Persistence using Local Storage
-  Filter Tasks
-  All Tasks
-  Active Tasks

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/ToDo-App.git
```

### Open the Project

```bash
cd ToDo-App
```

### Run the Application

Open `index.html` in your web browser.

Or use the **Live Server** extension in Visual Studio Code.

---

##  Application Workflow

1. Enter a task.
2. Click **Add Task**.
3. View the task in the task list.
4. Edit or delete tasks as needed.
5. Mark tasks as completed.
6. Filter tasks using:
   - All
   - Active
   - Completed
7. Refresh the browser—your tasks remain saved automatically.
